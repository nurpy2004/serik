Bot Prefix

The default prefix for ser_bot is "!" or "/"You can use this prefix before each command to interact with the bot (e.g., /poll).
Available Commands
Poll Command

    Usage: /poll
    Description: Start a new poll in your server. You can customize the question, set multiple voting options, and more.

Poll List Command

    Usage: /polllist
    Description: View a list of all open polls in your server.

Poll Help Command

    Usage: /pollhelp
    Description: Display a list of all available commands and their descriptions.

Close Poll Command

    Usage: /closepoll
    Description: Close a poll by providing the poll ID.

Translate Command

    Usage: /translate
    Description: Translate text to a specified language. Provide the text to translate and the target language.

Command Usage Examples
Poll Command Example

plaintext

/poll
  --question "What's your favorite color?"
  --multiple-options true
  --answer1 "Red"
  --answer2 "Blue"
  --answer3 "Green"

Translate Command Example

plaintext

/translate
  --text "Hello, how are you?"
  --target-language "es"
